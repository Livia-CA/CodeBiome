Sobre

Bem-vindos a um jogo totalmente iterativo, que mescla as mais variadas caracteristias dos principais biomas do Brasil com a magia que o CSS proporcina. Aqui você aprenderá conceitos de CSS enquanto explora a diversidade dos biomas e problemas que permeiam nos dias atuais.

Este trabalho foi desenvolvido para a matéria de Informática na Educação, ministrado pela professora Priscila Marques Kai, pela UFGD. Utilizamos como tecnologias para desenvolvimento da interface, HTML, CSS e JS. Bem como Figma, para criação de protótipo, Photoshop e Illustrator para desenvolimento de personagens e cenário. Já para a animação, foi utilzado primordialmente After Effects.
